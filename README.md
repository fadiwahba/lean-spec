# lean-spec v3

A thin Claude Code plugin that enforces a spec-driven development lifecycle — spec → implement → review → close — with hard phase-gate hooks and two-model cost arbitrage.

## Quick Install

```bash
claude --plugin-dir /path/to/lean-spec
```
